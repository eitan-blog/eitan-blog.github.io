什么是SPI

SPI 全程叫 Service Provider Interface 服务提供接口，它可以通过一个指定的接口 / 抽象类，寻找到预先配置好的实现类（并创建实现类对象）。

Spring 通过 SpringFactoriesLoader 读取放在 META-INF 目录下名为 spring.factories 的文件，获取文件中的权限定类名转化为实体对象。
