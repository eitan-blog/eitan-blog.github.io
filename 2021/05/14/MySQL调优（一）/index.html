查看线程连接个数 SHOW PROCESSLIST

  • id：表示session id
  • user：表示操作的用户
  • host：表示操作的主机
  • db：表示操作的数据库
  • command：表示命令类型
  • info：表示详细的sql语句
  • time：表示相应命令执行时间
  • state：表示命令执行状态
